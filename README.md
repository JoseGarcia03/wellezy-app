## Instalación

1. Clona el repositorio:

    ```bash
    git clone https://github.com/JoseGarcia03/wellezy-app.git
    ```

2. Navega al directorio del proyecto:

    ```bash
    cd tu-repositorio
    ```

3. Instala las dependencias:

    ```bash
    npm install
    ```
